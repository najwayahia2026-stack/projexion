# كيفية تشغيل خدمة AI

## طريقة سريعة (موصى بها)

### على Windows:

1. افتح مجلد `python_ai_service`
2. انقر نقراً مزدوجاً على ملف `start_ai_service.bat`
3. انتظر حتى تظهر رسالة "Starting AI Service..."

### على Linux/Mac:

1. افتح Terminal
2. اذهب إلى مجلد `python_ai_service`:
   ```bash
   cd python_ai_service
   ```
3. شغّل الملف:
   ```bash
   chmod +x start_ai_service.sh
   ./start_ai_service.sh
   ```

## طريقة يدوية

1. افتح Terminal/Command Prompt
2. اذهب إلى مجلد `python_ai_service`:
   ```bash
   cd python_ai_service
   ```
3. ثبّت المتطلبات (في المرة الأولى فقط):
   ```bash
   pip install -r requirements.txt
   ```
4. شغّل الخدمة:
   ```bash
   python main.py
   ```

## التحقق من عمل الخدمة

بعد تشغيل الخدمة، افتح المتصفح على:
- http://localhost:8001/health

يجب أن ترى:
```json
{
  "status": "healthy",
  "service": "SmartGrad AI Similarity Service",
  "version": "2.0.0"
}
```

## ملاحظات مهمة

- **لا تغلق نافذة Terminal** أثناء استخدام الخدمة
- الخدمة تعمل على Port 8001
- يجب أن تبقى الخدمة قيد التشغيل أثناء استخدام النظام

## إيقاف الخدمة

اضغط `Ctrl+C` في نافذة Terminal

## اختبار الخدمة

بعد تشغيل الخدمة، يمكنك اختبارها:
```bash
python test_service.py
```

## حل المشاكل

إذا واجهت أي مشاكل، راجع ملف `INSTALL.md` للحلول التفصيلية.
