# دليل مشروع ProjexiOn — شرح المشروع والملفات

منصة ويب لإدارة مشاريع التخرج (Laravel + Python AI). هذا الملف يشرح هيكل المشروع ووظيفة المجلدات والملفات الرئيسية.

---

## 1. نظرة عامة على المشروع

| البند | الوصف |
|-------|--------|
| **اسم المشروع** | ProjexiOn (SmartGrad) |
| **الغرض** | إدارة مشاريع التخرج: طلاب، مشرفون، لجنة تقييم، إدارة قسم، مدير نظام |
| **Backend** | Laravel (PHP 8.2+) |
| **Frontend** | Blade + Tailwind CSS 4 + Vite + JavaScript |
| **قاعدة البيانات** | MySQL / MariaDB |
| **خدمة إضافية** | Python (FastAPI) لفحص التشابه واكتشاف المحتوى المولد بالذكاء الاصطناعي |

## 3. تسجيل الدخول والحسابات التجريبية

بعد تشغيل الـ seeders (`php artisan db:seed` أو `QuickLoginSeeder`):

| الدور | البريد | كلمة المرور |
|-------|--------|-------------|
| مدير النظام | `admin@smartgrad.com` | `password` |
| طالب | `student@smartgrad.com` | `password` |
| مشرف | `supervisor@smartgrad.com` | `password` |
| لجنة التقييم | `committee@smartgrad.com` | `password` |

- تسجيل دخول المستخدمين: `/login`
- تسجيل دخول الإدارة: `/admin`

## 4. التشغيل السريع

```bash
# إعداد أول مرة
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan db:seed

# حزم
composer install
npm install && npm run build

# تشغيل Laravel
php artisan serve
# التطبيق: http://127.0.0.1:8000

# تشغيل خدمة AI (في مجلد آخر أو طرفية ثانية)
cd python_ai_service
pip install -r requirements.txt
python main.py
# الخدمة: http://localhost:8001
```

على Linux/Mac يمكن استخدام: `./start_project.sh` (بعد `chmod +x start_project.sh`).
